Markdown 语法详细教学文档

本文档详细介绍 Markdown 语法，从基础到高级特性，帮助您快速上手并深入理解 Markdown 的强大功能。

1. Markdown 简介

Markdown 是一种轻量级标记语言，可以让你使用纯文本编写格式化内容，并轻松转换为 HTML。它被广泛用于编写文档、博客、README 文件等。

2. 基础语法

2.1 标题

使用 # 来表示标题，等级从 1（最大）到 6（最小）。

2.2 段落与换行

  • 段落通过一个或多个空行来分隔。
  • 换行：在行尾添加两个或以上空格，或使用 <br> 标签。

2.3 强调文字

  • 斜体：用单个 * 或 _ 包围文本。
  • 粗体：用两个 * 或 _ 包围文本。
  • 粗斜体：用三个 * 混合使用。

2.4 列表

无序列表

使用 -、* 或 + 表示无序列表项。

有序列表

前面使用数字和点号标记。

2.5 链接与图片

链接

格式为 [链接文本](URL)。

图片

格式为 ![替代文本](图片URL)。

2.6 引用与代码块

引用

使用 > 创建引用内容。

行内代码

用单个反引号 ` 包含代码。

多行代码块

用三个反引号围住多行代码，并可指定高亮语言。

2.8 分隔线

使用三个或以上的 -、* 或 _ 来创建分隔线。

3. 高级语法

3.1 任务列表

对于要跟踪任务或完成情况的列表，可使用任务列表语法。

3.2 代码高亮与行号

许多 Markdown 编辑器支持代码高亮。指定语言有助于提高代码的可读性。有时也支持显示行号，但这通常依赖于具体平台的功能。

3.4 脚注

部分 Markdown 渲染器支持脚注，可以用于提供额外说明。

3.5 定义列表

某些 Markdown 扩展支持定义列表格式。

3.6 缩略语

一些扩展允许定义缩略语，以便在文档中统一解释。

在正文中使用 HTML 时，HTML 自动变为带有说明的缩略语。

3.7 表情符号 (Emoji)

许多 Markdown 渲染器允许使用 Emoji 表情，通过 :emoji_name: 语法。

3.8 自动链接

当你在文本中直接写出 URL 时，大多数 Markdown 渲染器会自动将其转换为链接。

3.9 内嵌 HTML

Markdown 允许嵌入 HTML 代码，以便实现更复杂的布局和样式。

3.10 折叠内容（可展开/折叠部分）

某些平台支持折叠内容，可以用 HTML 的 <details> 标签实现。

4. 插件和扩展

不同的平台（如 GitHub、Jekyll、Hugo 等）可能支持额外的 Markdown 扩展。以下是一些常见的扩展：

  • Pandoc: 支持幻灯片、书籍等多种格式转换。
  • Markdown-it: 一个健壮的 Markdown 解析器，可插件化扩展功能。
  • Kramdown: Ruby 编写的 Markdown 解析器，支持多种扩展语法。

5. 总结

本文档全面讲解了 Markdown 的基础语法与高级特性。无论是日常笔记编写、文档撰写还是复杂网页内容构建，Markdown 都能提供简单而高效的解决方案。掌握这些语法后，不妨探索各平台自定义扩展及插件，发挥 Markdown 的无限可能。

Happy Markdowning!
